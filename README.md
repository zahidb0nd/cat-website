# Hussain Cattery Website

This is the official website for Hussain Cattery, a premier cattery based in Bangalore, specializing in ethical breeding of high-quality kittens.

## About the Project

This project is a Next.js application designed to showcase the cattery's kittens, provide information about their breeding practices, and facilitate contact with potential families. It features a modern, responsive design with smooth animations and interactive elements.

## Features

- **Hero Section**: Engaging introduction to the cattery.
- **Trust & Quality Assurance**: Highlights key features like Health Checked, TICA Registered, Home Raised, and Global Shipping.
- **Kitten Gallery**: A visual showcase of available kittens.
- **Cattery Moments**: A glimpse into the life at the cattery.
- **Contact & Enquiries**: Easy ways to reach out via WhatsApp, Email, or visit the location.
- **FAQ Section**: Answers to common questions about adoption, health, and shipping.
- **Responsive Design**: Optimized for all devices, from mobile phones to desktops.
- **Smooth Animations**: Uses Framer Motion for delightful user interactions.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: Custom components and [Radix UI](https://www.radix-ui.com/) primitives.

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/cats.git
   cd cats
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Contains the application routes and pages.
- `src/components`: Reusable UI components.
  - `ui`: Base UI components (buttons, navbar, footer, etc.).
  - `Hero.tsx`, `KittenGallery.tsx`, etc.: Section-specific components.
- `src/lib`: Utility functions and shared logic.
- `public`: Static assets (images, fonts, etc.).
